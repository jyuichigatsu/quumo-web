## 5. 注意事項

1. **デザインガイドライン**
   - カラースキーム: メインカラー #007AFF（青）
   - フォント: Noto Sans JP
   - アイコン: Material Icons
   - 余白: 16px（モバイル）、24px（デスクトップ）

2. **アクセシビリティ**
   - コントラスト比: 4.5:1以上
   - フォントサイズ: 最小14px
   - タップ領域: 最小44x44px

3. **パフォーマンス**
   - 画像の最適化
   - レイジーローディングの実装
   - キャッシュ戦略の検討
